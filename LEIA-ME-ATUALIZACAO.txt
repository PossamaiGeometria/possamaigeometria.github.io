COMO PUBLICAR NO GITHUB

1. Descompacte o arquivo recebido.
2. Abra o repositório:
   PossamaiGeometria/possamaigeometria.github.io
3. Clique em Add file.
4. Clique em Upload files.
5. Arraste TODOS os arquivos e pastas que estão dentro da pasta descompactada
   para a área de upload do GitHub.
6. Confirme que aparecem, entre outros:
   - /alinhamento-de-maquinas/index.html
   - /es/alineacion-de-maquinas/index.html
   - /assets/service-page.css
   - /index.html
   - /es/index.html
   - /sitemap.xml
7. No campo da mensagem, use:
   Nova página de alinhamento de máquinas
8. Mantenha selecionado:
   Commit directly to the main branch
9. Clique em Commit changes.

SEARCH CONSOLE

Depois que as páginas abrirem corretamente:

1. Abra o Google Search Console.
2. Use Inspeção de URL.
3. Cole:
   https://possamaigeometria.com.br/alinhamento-de-maquinas/
4. Clique em Testar URL publicado.
5. Se o teste estiver correto, clique em Solicitar indexação.
6. Repita para:
   https://possamaigeometria.com.br/es/alineacion-de-maquinas/
7. Em Sitemaps, confirme que continua cadastrado:
   https://possamaigeometria.com.br/sitemap.xml
